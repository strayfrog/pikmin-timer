⚙️ PWA 自助打包與離線配置器

Progressive Web App (PWA) 技術可以讓您的網頁完美安裝至手機主畫面，並且擁有完全獨立的 App 視窗、圖示與高度防休眠特性。 因為瀏覽器的限制，PWA 安裝需要將主網頁 (index.html) 與服務線程式 (sw.js) 放置於同個資料夾中部署。請點擊下方按鈕分別下載這兩個檔案：

📄 1. 主網頁程式 (index.html)

包含主計算面板與介面樣式，是主要的應用程式檔。

⚙️ 2. 服務線程式 (sw.js)

滿足 PWA 離線標準的背景服務，是觸發手機彈出「安裝 App」提示的必備檔案。

🚀 零門檻免費網頁架設教學

  1. 下載上方的兩個檔案： index.html 與 sw.js。
  2. 在電腦上建立一個空資料夾，把這兩個檔案都放進去。
  3. 打開瀏覽器，造訪免費拖放架站平台 Netlify Drop (https://app.netlify.com/drop)。
  4. 將剛剛的資料夾直接拖曳丟進 Netlify Drop 網頁正中間的虛線框內。
  5. 架設完成！您會獲得一個專屬的雲端網址（例如：https://xxxx.netlify.app）。

📲 手機主畫面安裝步驟 (PWA 模式)

🍏 iPhone / Safari 安裝

1. 用 iPhone 內建的 Safari 開啟您的專屬網址。
2. 點擊下方的「分享」按鈕（正方形往上箭頭）。
3. 往下滑選單，選擇「加入主畫面」，點擊右上角新增。
4. 回到桌面，點開可愛蘑菇圖示，即可以乾淨的 App 全螢幕模式執行。

🤖 Android / Chrome 安裝

1. 用 Chrome 開啟您的專屬網址。
2. 當偵測到 sw.js 運作時，螢幕下方會自動彈出「將皮克敏計算器加到主畫面」的提示。
3. 或點擊右上角「三個點」，選擇「安裝應用程式」或「加入主畫面」。
